Política de Privacidad

En Automatización de video n8n respetamos la privacidad de nuestros usuarios. Esta política explica qué datos recopilamos, con qué fin y cómo los protegemos.

Información que recopilamos

  • Información de autenticación necesaria para conectar la cuenta de TikTok (OAuth tokens).
  • Contenido que el usuario selecciona para publicar (videos, textos, descripciones).
  • Datos técnicos como registros de uso y direcciones IP para fines de diagnóstico.

Uso de la información

Utilizamos la información exclusivamente para facilitar la creación, montaje y publicación de contenidos en TikTok según las instrucciones del usuario.

Compartición de datos

No compartimos datos de usuario con terceros salvo cuando sea necesario para el funcionamiento (por ejemplo, proveedores de TTS o servicios de renderizado). En dichos casos se aplican políticas de protección de datos.

Conservación

Los datos se conservan durante el tiempo necesario para proporcionar el servicio o mientras la cuenta del usuario esté activa.

Seguridad

Aplicamos medidas razonables de seguridad para proteger los datos contra accesos no autorizados.
