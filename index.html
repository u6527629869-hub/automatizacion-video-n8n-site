Automatización de video n8n

Esta aplicación automatiza la generación y publicación de vídeos en TikTok usando flujos de trabajo en n8n. Se integra con servicios de generación de guiones (IA), TTS, generación de imágenes y la API de TikTok para subir y programar publicaciones.
